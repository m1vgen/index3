1. Cechy modyfikowania formatowania w HTML:
HTML umożliwia podstawowe formatowanie treści, takie jak:
- Strukturalne elementy (nagłówki, akapity, listy).
- Wstawianie obrazków i linków.
- Tworzenie tabel.

2. Nowe możliwości CSS w porównaniu z HTML:
CSS wprowadza zaawansowane możliwości formatowania, takie jak:
1. Precyzyjne układy (flexbox, grid).
2. Stylizacja elementów (czcionki, kolory, tła).
3. Animacje i przejścia.
4. Pozycjonowanie elementów (absolute, fixed).
5. Responsywność (media queries).

3. Wady CSS:
 CSS ma też wady:
1. Trudności w utrzymaniu dużych projektów.
2. Problemy z kompatybilnością między przeglądarkami.
3. Możliwe problemy z wydajnością przy animacjach.
4. Trudności w precyzyjnym pozycjonowaniu elementów.
5. Starsze przeglądarki nie wspierają nowych funkcji CSS.
